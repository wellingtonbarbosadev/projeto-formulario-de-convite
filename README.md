# Projeto Formulário de Convite

Este projeto foi desenvolvido como parte de um desafio do curso da Rocketseat, com o objetivo de praticar e consolidar conhecimentos em desenvolvimento web moderno.

## Sobre o projeto
O sistema consiste em um formulário de criação de convites para eventos, com layout responsivo e foco em experiência do usuário. O desafio envolveu criar uma interface intuitiva, estilizada e funcional, utilizando boas práticas de HTML e CSS.

## Tecnologias utilizadas
- **HTML5**
- **CSS3** (incluindo variáveis CSS, grid, flexbox, transições e animações)

## Estrutura de pastas
```
├── index.html
├── assets/
│   ├── css/
│   │   ├── global.css
│   │   ├── index.css
│   │   ├── layouts.css
│   │   └── forms/
│   │       └── inputs.css
│   ├── icons/
│   └── img/
```

## Como executar
Abra o arquivo `index.html` em seu navegador. Não é necessário backend ou dependências externas para rodar o projeto.

## Funcionalidades
- Formulário completo para criação de convites de eventos
- Campos de entrada com validação visual
- Seleção de temas para o evento
- Seleção de cores personalizadas
- Toggle switches customizados
- Radio buttons estilizados
- Design responsivo e moderno
